# Meeting Notes

Notas de reuniones y sincronizaciones del equipo.

## Formato

Nombrar archivos como: `YYYY-MM-DD-tema.md`

Ejemplo: `2026-03-24-sync-multiplayer.md`

## Plantilla

```markdown
# Sync — Tema

**Fecha**: YYYY-MM-DD
**Asistentes**: @usuario1, @usuario2

## Temas tratados

- Punto 1
- Punto 2

## Decisiones

- Decision 1 (responsable: @usuario)

## Action items

- [ ] Tarea 1 — @responsable
- [ ] Tarea 2 — @responsable

## Proxima reunion

YYYY-MM-DD
```
